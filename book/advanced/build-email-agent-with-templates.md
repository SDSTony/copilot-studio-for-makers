# 이메일 템플릿 기반으로 이메일을 발송하는 에이전트 구축

```{warning}
**Copilot Studio는 빠르게 발전하는 제품입니다.** UI 업데이트가 잦아 본 가이드의 스크린샷·메뉴 명칭과 실제 화면이 일부 다를 수 있습니다. 또한 SaaS 제품의 특성상 새로운 기능이 순차적으로 출시되는 과도기에는 사용자·환경마다 화면 구성이 조금씩 다르게 보일 수 있습니다.
```

## HTML이란?

웹 페이지나 이메일을 열었을 때 글자가 굵게 표시되고, 색이 들어가고, 표나 이미지가 깔끔하게 정렬되어 보이는 이유는 **HTML** 이라는 언어 덕분입니다.

HTML은 쉽게 말해 **"이 글자는 빨간색으로, 이 부분은 표로, 여기는 큰 제목으로 보여줘"** 라고 컴퓨터에게 알려주는 언어입니다. 실제로는 아래와 같이 생겼습니다.

```html
<h1>안녕하세요!</h1>
<p style="color: red;">이 문장은 빨간색으로 표시됩니다.</p>
<p>일반 문장은 이렇게 보입니다.</p>
```

이 코드를 메일 본문에 넣으면, 수신자의 Outlook이나 Gmail에서 **실제 서식이 적용된 모습**으로 보여집니다.

---

## Outlook 이메일도 HTML로 만들어진다

사실 우리가 Outlook에서 이메일을 작성할 때, 화면에서는 Word처럼 편집하는 것처럼 보이지만 **뒤에서는 HTML 코드가 자동으로 만들어지고 있습니다.**

예를 들어, 글자를 굵게 하면 `<b>굵은 글자</b>`, 색을 바꾸면 `style="color: blue;"` 같은 코드가 자동으로 생성됩니다. 우리는 그걸 보지 못할 뿐입니다.

이 원리를 이용하면, **HTML을 전혀 몰라도** 이메일 템플릿을 만들 수 있습니다:

> 1. Outlook에서 보내고 싶은 이메일 양식을 **평소처럼 디자인**한다
> 2. 그 이메일의 **HTML 코드를 복사**해온다
> 3. Power Automate의 **이메일 발송 커넥터에 붙여넣는다**
> 4. 이름, 날짜처럼 **매번 바뀌는 부분만 변수로 교체**한다

Outlook이 HTML을 대신 만들어주는 셈이므로, 코딩 없이 예쁜 이메일 템플릿을 에이전트에 연결할 수 있습니다.

---

## Outlook에서 HTML 소스 복사하는 방법

```{note}
아래 방법은 **Outlook Classic 데스크톱 앱(Windows)** 기준입니다.
```

**① 템플릿으로 사용할 이메일 작성하기**

새 메일 작성 화면에서 보내고 싶은 형태의 이메일을 평소처럼 꾸며줍니다. 회사 로고 삽입, 표 추가, 색상 강조 등 원하는 서식을 모두 적용한 뒤, 일단 임시 저장하거나 자신에게 발송합니다.

**② 이메일을 더블 클릭해서 열기**

받은 편지함에서 해당 이메일을 **더블 클릭**하여 별도 창으로 엽니다.

**③ Actions 메뉴 → View Source 클릭**

열린 메일 창 상단 메뉴에서 **Actions → View Source** 를 클릭합니다. 그러면 메모장(Notepad)이 자동으로 열리면서 해당 이메일의 HTML 코드가 표시됩니다.

**④ HTML 코드 전체 복사**

메모장에서 **Ctrl + A** 로 전체 선택한 뒤, **Ctrl + C** 로 복사합니다.

```{tip}
코드가 복잡해 보여도 괜찮습니다. 전부 이해할 필요 없이 **복사해서 붙여넣는 것**이 핵심입니다.
```

## 이메일 발송 커넥터에 HTML 붙여넣기

Power Automate에서 **"이메일 보내기(V2)"** 액션을 사용하면 HTML 형식의 이메일을 보낼 수 있습니다.

**① Body 필드를 HTML 모드로 전환하기**

이메일 보내기 액션의 **본문(Body)** 필드 오른쪽에 있는 `</>` 버튼을 클릭합니다. 이 버튼을 누르면 일반 텍스트 입력 모드에서 **HTML 코드 입력 모드**로 바뀝니다.

```{warning}
`</>` 버튼을 누르지 않고 HTML 코드를 붙여넣으면, 태그 기호들이 그대로 텍스트로 노출됩니다. 반드시 HTML 모드로 먼저 전환하세요.
```

**② 복사한 HTML 붙여넣기**

HTML 모드로 전환된 Body 필드에 아까 복사한 HTML 코드를 붙여넣습니다.

**③ 매번 바뀌는 부분을 변수로 교체하기**

HTML 코드 안에서 **고정된 이름, 날짜, 내용 등을 변수로 바꿔줍니다.** 예를 들어, 코드 안에 `홍길동` 이라고 하드코딩된 부분을 Copilot Studio에서 받아온 이름 변수로 교체하면, 사람마다 다른 이름으로 이메일이 발송됩니다.

```html
<!-- 변경 전: 이름이 고정되어 있음 -->
<p>안녕하세요, 홍길동 님.</p>

<!-- 변경 후: 변수로 교체 -->
<p>안녕하세요, [사용자이름] 님.</p>
```

---

## 전체 흐름 요약

```
① 사용자가 에이전트에게 정보 입력 (이름, 날짜, 내용 등)
         ↓
② Copilot Studio가 Power Automate 플로우 호출
         ↓
③ Power Automate가 HTML 템플릿에 변수 삽입
         ↓
④ Office 365 Outlook 커넥터로 이메일 발송
         ↓
⑤ 수신자가 브랜딩된 HTML 이메일 수신
```

이 흐름을 통해 **사용자가 에이전트에 정보를 입력하면, 자동으로 예쁘게 꾸며진 이메일이 발송**되는 에이전트를 구축할 수 있습니다.
